# Unit Economics

## Exemples d’unités
- coût par domaine produit
- coût par environnement
- coût par namespace
- coût par transaction simulée
- coût d’un workload stateful critique
- coût d’un pipeline Data/IA
- coût d’un service d’inférence
- coût de stockage vectoriel par domaine

## Lecture recommandée
Les unités économiques doivent être relues avec trois angles :
- valeur métier ;
- criticité ;
- soutenabilité opérationnelle.

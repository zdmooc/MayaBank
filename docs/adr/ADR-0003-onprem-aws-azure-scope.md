# ADR-0003 — On-Prem + AWS + Azure Scope

## Statut
Accepted

## Décision
Le périmètre provider est limité à **on-prem, AWS et Azure**.

## Conséquences
- le dépôt reste dense mais maîtrisable ;
- GCP est explicitement hors périmètre initial ;
- le multi-cloud reste crédible sans devenir incontrôlable.

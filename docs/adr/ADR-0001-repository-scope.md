# ADR-0001 — Repository Scope

## Statut
Accepted

## Décision
MayaBank sera un dépôt **documentaire + architecture + lab semi-exécutable + gouvernance + FinOps**.

## Conséquences
- le dépôt ne sera pas limité à des manifestes techniques ;
- il intégrera HLD, LLD, ADR, runbooks, sécurité, FinOps et gouvernance ;
- il devra rester lisible et reprenable dans le temps.

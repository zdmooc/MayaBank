# ADR-0002 — OKD as Open Source Base

## Statut
Accepted

## Décision
La base pédagogique et technique du dépôt sera **OKD / Kubernetes open source**.

## Motivation
- cohérence avec un objectif 100% open source ;
- proximité conceptuelle avec une cible OpenShift enterprise ;
- excellent terrain d'apprentissage du socle plateforme.

## Conséquences
- le dépôt documentera explicitement le mapping OKD → OpenShift enterprise ;
- la production réelle n'est pas simulée telle quelle ;
- certaines briques enterprise resteront des équivalents conceptuels.

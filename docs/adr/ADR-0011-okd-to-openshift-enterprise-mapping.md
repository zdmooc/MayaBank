# ADR-0011 — OKD to OpenShift Enterprise Mapping

## Statut
Accepted

## Décision
Le dépôt doit documenter explicitement la translation entre le modèle MayaBank basé sur OKD et une cible OpenShift enterprise.

## Conséquences
- différence entre apprentissage open source et cible enterprise rendue visible ;
- le dépôt reste crédible pour un objectif Architecte OpenShift Senior ;
- la trajectoire vers des briques enterprise est claire.

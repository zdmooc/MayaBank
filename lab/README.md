# Lab

Le répertoire `lab/` contient des éléments **semi-exécutables** ou préparatoires.

## Objectifs
- disposer d'une base locale crédible ;
- documenter les hypothèses on-prem / AWS / Azure ;
- fournir des exemples de namespaces, quotas et network policies ;
- garder la séparation entre lab, architecture et cible future.

## Niveaux
- `local/` : exemples rejouables ou proches du rejouable
- `onprem/` : hypothèses de topologie et d'inventaire
- `aws/` : variables et hypothèses
- `azure/` : variables et hypothèses
